🔑 Groq API Key (GRATIS)
Diperlukan untuk fitur IMEI Lookup dan Phone Lookup. Key tersimpan hanya di browser kamu, tidak dikirim ke mana pun selain Groq.
Cara dapat Groq API Key (GRATIS):
1. Buka console.groq.com/keys
2. Daftar akun gratis (bisa pakai Google)
3. Klik Create API Key → beri nama → copy key-nya
4. Paste di kolom di atas → klik Simpan & Test

✅ Groq GRATIS hingga 14.400 request/hari — lebih dari cukup!
